# 🚀 MotionLab

MotionLab is an interactive projectile motion simulator built with HTML, CSS and JavaScript.

The application allows users to explore how velocity, launch angle and gravity affect projectile trajectories through real-time physics calculations and animated visualization.

---

## ✨ Features

- 🌍 Planet selection system
- 📈 Height × Time graph
- ➡️ Real-time velocity vectors
- 💫 Glow trajectory rendering
- 🎯 Adjustable velocity and angle
- ⚡ Real-time simulation
- 📊 Flight time, range and maximum height calculations

---

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
- Canvas API

---

## 🎓 Physics Concepts

- Projectile motion
- Kinematics
- Velocity decomposition
- Gravity variation
- Trajectory analysis
